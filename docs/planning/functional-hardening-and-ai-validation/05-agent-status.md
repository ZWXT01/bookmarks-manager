## 1. 自动执行规则

- 本文件只用于 issue 编码执行模式。
- 风险排查执行模式应改为优先读取 [风险台账](./06-risk-log.md)。
- Agent 必须优先读取本文件。
- Agent 必须选择第一个依赖已满足且 `status=todo` 的任务。
- 开始前写入 `started_at`。
- 完成后写入 `completed_at`。
- 阻塞时写入 `blocked_reason`。
- 每个任务完成后，必须额外留下 1 次独立 Git commit。
- 若当前目录还不是 Git 仓库，则必须先初始化仓库。
