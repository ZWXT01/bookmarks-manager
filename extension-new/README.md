# 书签管理器浏览器扩展

一个用于保存书签和网页快照的 Chrome/Edge 浏览器扩展。

## 功能

- 📚 **保存书签** - 一键保存当前网页到书签管理器
- 📸 **保存快照** - 使用 SingleFile 保存网页完整副本
- 🔗 **书签+快照** - 同时保存书签和快照
- 🔍 **链接检测** - 检查网址是否可访问
- 📂 **分类管理** - 支持选择书签分类

## 安装

### 从源码安装（开发者模式）

1. 克隆或下载项目
2. 打开 Chrome/Edge 浏览器，访问 `chrome://extensions`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `extension-new` 目录

## 配置

首次使用需要配置服务器连接：

1. 点击扩展图标打开弹窗
2. 展开"⚙️ 设置"
3. 填写服务器地址（如 `http://localhost:8080`）
4. 填写 API Token（在书签管理器设置页面获取）
5. 点击"保存设置"

## 使用方法

### 保存书签
1. 打开要保存的网页
2. 点击扩展图标
3. 可选择分类
4. 点击"📌 保存书签"

### 保存快照
1. 打开要保存的网页
2. 点击扩展图标
3. 点击"📸 保存快照"
4. 等待处理完成（可能需要几秒到几分钟）

### 书签+快照
一键同时保存书签和快照。

## 技术说明

- 使用 Manifest V3
- 集成 SingleFile 用于网页快照
- 支持 Chrome 和 Edge 浏览器

